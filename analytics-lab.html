📂 FAQ Accordion

What is Google Tag Manager? +
GTM is a tag management system that lets you deploy marketing and analytics tags without editing code. Each accordion open fires an accordion_open event with the question as a dimension — helping you see which FAQs get the most engagement.
How does GA4 differ from Universal Analytics? +
GA4 is event-based (vs. session-based), uses machine learning, and supports cross-platform tracking. Track your accordion interactions to understand which FAQs users explore on their path to conversion.
What are GA4 custom dimensions & metrics? +
Custom dimensions let you attach extra data to events — like user_role, plan_name, or page_section. You can register these in GA4 Admin → Custom Definitions.
What is the difference between GTM Triggers and Tags? +
Tags are what fires (GA4 event, Facebook Pixel, etc.). Triggers are when they fire (page view, click, form submit). Variables provide the dynamic data passed with each tag.
